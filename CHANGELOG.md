# Changelog

## 1.0.0

- Initial production release
- Android App Links support (verified HTTPS)
- iOS Universal Links support
- Custom URI scheme deep links
- Deferred deep link recovery (no Firebase required)
- Dynamic route matching with named path parameters
- Link generator (HTTPS, custom scheme, QR-compatible)
- Analytics adapter interface with console adapter built-in
- Firebase Hosting configuration with optional Cloud Functions
- Web redirect page with App Store / Play Store fallback
- Full test suite
